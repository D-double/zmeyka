# Змейка (Neon Snake)

Классическая игра «Змейка» в неоновом стиле. Написана на чистом HTML, CSS и JavaScript (Canvas).

## Как играть

- Управляйте змейкой с помощью **клавиш-стрелок** (↑ ↓ ← →) на клавиатуре
- На мобильных устройствах — **свайпы** (проведите пальцем в нужную сторону)
- Собирайте розовую еду, чтобы увеличить счёт (+10 очков) и длину змейки
- Игра заканчивается, если змейка врезается в стену или в собственное тело

## Особенности

- Неоновая графика с эффектом свечения (CSS + Canvas)
- Скорость игры постепенно увеличивается (каждые 10 очков)
- Рекорд сохраняется в `localStorage` между сессиями
- Адаптивный дизайн (работает на телефонах и планшетах)
- Оверлей «Игра окончена» с возможностью начать заново

## Файлы

- `index.html` — вся игра в одном файле (HTML, стили, скрипт)

## Запуск

Откройте `index.html` в любом современном браузере.
